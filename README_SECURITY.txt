Cach dung:
1. Mo GitHub repo otp24h.
2. Xoa toan bo noi dung file server.js cu, copy noi dung file server.js moi vao.
3. Xoa toan bo noi dung file App.tsx cu, copy noi dung file App.tsx moi vao.
4. Commit va deploy lai Render/Vercel.
5. Tren Render them ENV neu can reset admin:
   ADMIN_USERNAME=admin
   ADMIN_PASSWORD=azhung12
   RESET_ADMIN_PASSWORD=true
   REQUEST_TIMEOUT_MS=8000
6. Deploy xong dang nhap admin / azhung12.
7. Sau khi vao duoc admin, doi RESET_ADMIN_PASSWORD=false roi deploy lai.
